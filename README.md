# CrowdLend

Peer-to-peer micro-lending platform.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/CrowdLend.sol`
- Stacks: `contracts/stacks/contracts/crowdlend.clar`

## Setup
```bash
pnpm install
pnpm dev
```
